# triproject

**项目说明** 
- 采用Spring,SpringMVC,MyBatis,Shiro框架，开发的一套权限系统，极低门槛，拿来即用。
<br>
技术栈：springMVC+spring+shiro+mybatis+jquery+bootstrap+mysql
